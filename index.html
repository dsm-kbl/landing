Welcome to Cloud Infrastructure!

An SKF cloud project has dramatically increased maintenance efficiency and reliability for pulp and paper manufacturer Stora Enso, helping the customer to achieve significant savings. In recognition of this impressive feat, SKF has won in the hotly contested 'Cloud' category at this year's Software AG Innovation Awards.

SKF wins award for cloud innovation

The SKF team working with Stora Enso saw that it could increase efficiency if work notifications could be automatically created in the customer's enterprise resource planning (ERP) system whenever SKF's @ptitude Analyst software finds condition-based data in need of special attention.
